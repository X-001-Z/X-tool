# Contributing

Contributions are welcome.

1. Create a branch for the change.
2. Keep changes focused and include tests for behavior changes.
3. Run `python -m unittest -v test_pdf2ppt.py`.
4. Do not commit PDFs or PPTX files containing confidential information.
5. Explain user-visible changes in `CHANGELOG.md`.
