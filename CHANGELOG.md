# Changelog

## 1.5.0 - 2026-06-11

- Replaced PyMuPDF with PDFium through pypdfium2.
- Removed the AGPL dependency from the distributed application.
- Reduced the Windows executable from approximately 40.8 MB to 24.2 MB.
- Preserved batch conversion, password PDF support, mixed-page handling, cancellation, safe output naming, and DPI controls.
- Added reproducible dependency files and public-release documentation.

## 1.4.1 - 2026-06-11

- Fixed the bottom action buttons being hidden at common Windows display scaling levels.
- Added DPI-aware initial and minimum window sizing.

## 1.4.0 - 2026-06-10

- Added batch conversion, quality presets, cancellation, password PDF support, atomic saving, safe output naming, and large-page protection.
- Fixed command-line DPI handling and mixed-page stretching.
